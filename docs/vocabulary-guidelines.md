# Kana Master 例詞選擇規範

例詞的目的不是展示最完整的詞彙，而是幫助繁體中文使用者快速建立聲音與假名的連結。

## 優先順序

1. 台灣使用者普遍熟悉的日本文化詞：壽司、拉麵、動畫、遊戲、寶可夢等。
2. 日本旅遊高頻詞：車站、車票、電車、飯店、廁所、百貨公司等。
3. 生活中容易想像的具體名詞：貓、狗、蘋果、牛奶、蛋糕等。
4. 動漫與遊戲常見詞：世界、角色、龍、猜拳等。
5. 只有在缺乏自然替代詞時，才採用低頻或擬聲詞。

## 編輯原則

- 例詞應以目標音節開頭；`ぢ`、`づ` 等通常不在詞首的音節除外。
- 優先選一眼就能形成畫面的詞，避免抽象、專業或需要額外文化說明的詞。
- 外來語可使用片假名，讓學習者同時建立片假名辨識。
- 不為了熟悉度犧牲正確性；品牌與角色名稱需使用通行日文寫法。
- 同一列盡量涵蓋生活、旅遊、飲食與流行文化，避免全部來自單一情境。
- 罕見拗音若沒有常用詞，選擇真實存在、發音清楚且相對容易搜尋或解釋的詞。

## 審查問題

新增或替換例詞時，依序確認：

1. 台灣初學者是否可能在學日文前就聽過？
2. 去日本旅遊時是否可能看到或用到？
3. 是否能立即形成具體畫面？
4. 目標假名是否位於詞首或容易被聽出？
5. 是否有更自然、更常見且同樣符合目標音的替代詞？
